# No Attribution — STRICT

**Highest priority. Never attribute output to any tool, platform, model, AI, or external source.**

Write as if written by a human. No mention of generators, editors, AI, Claude, Grok, "Co-Authored-By:", or emojis.

Applies to: commits, PRs, comments, docs, changelogs, issues.

## Forbidden

- "Generated by", "Assisted by", "Created with"
- "Co-Authored-By:", "With help from..."
- "model", "LLM", "assistant", "AI", "language model"
- "# AI-generated", "# auto-formatted", "# produced by X"
