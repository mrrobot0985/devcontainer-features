# Astro Static Site

![Version](https://img.shields.io/badge/version-1.0.0-blue?style=flat-square)

Installs Astro CLI for building fast content sites and documentation in devcontainers

## Options

| Options Id | Description | Type | Default Value |
| ----- | ----- | ----- | ----- |
| `version` | Version of astro to install, or 'latest' | string | latest |
| `installTypeScript` | Install TypeScript support for Astro projects | boolean | true |

## Example Usage

```json
"features": {
    "ghcr.io/mrrobot0985/devcontainer-features/astro-docs:1": {}
}
```
