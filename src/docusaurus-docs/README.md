# Docusaurus Documentation Site

![Version](https://img.shields.io/badge/version-1.0.0-blue?style=flat-square)

Installs Docusaurus for building documentation websites in devcontainers

## Options

| Options Id | Description | Type | Default Value |
| ----- | ----- | ----- | ----- |
| `version` | Version of @docusaurus/core to install, or 'latest' | string | latest |
| `initTemplate` | Template to use when initializing a new Docusaurus site | string | classic |

## Example Usage

```json
"features": {
    "ghcr.io/mrrobot0985/devcontainer-features/docusaurus-docs:1": {}
}
```
