# Bruno API Testing

![Version](https://img.shields.io/badge/version-1.0.0-blue?style=flat-square)

Installs Bruno CLI for local-first, Git-friendly API testing and collection management in devcontainers

## Options

| Options Id | Description | Type | Default Value |
| ----- | ----- | ----- | ----- |
| `version` | Version of @usebruno/cli to install, or 'latest' | string | latest |
| `globalInstall` | Install Bruno CLI globally via npm | boolean | true |

## Example Usage

```json
"features": {
    "ghcr.io/mrrobot0985/devcontainer-features/bruno-api-testing:1": {}
}
```
